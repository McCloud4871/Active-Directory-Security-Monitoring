# Active Directory Security Monitoring Lab
Built a virtual enterprise lab environment consisting of Active Directory, Windows 10, Splunk Enterprise, Kali Linux, Sysmon, and Splunk Universal Forwarders. The project demonstrates centralized log collection, Active Directory administration, attack simulation, and security monitoring using Splunk.
## Skills Demonstrated

- Active Directory Administration
- Windows Server 2022
- Splunk Enterprise
- Splunk Universal Forwarder
- Sysmon Configuration
- Windows Event Logging
- VirtualBox Networking
- Security Monitoring
- Threat Detection
- MITRE ATT&CK
- Atomic Red Team
- Authentication Event Analysis

## Security Findings

### Failed Authentication Detection

Event ID 4625 was collected from the target workstation and forwarded to Splunk. Multiple failed logon attempts generated by Hydra were identified.

### Successful Authentication Detection

Event ID 4624 confirmed successful authentication after Hydra discovered the valid password.

### Atomic Red Team Detection

MITRE ATT&CK technique T1136.001 (Create Local Account) generated telemetry that was successfully collected and searched within Splunk.


## Lessons Learned

- Configured centralized log collection using Splunk Universal Forwarders.
- Learned how Sysmon improves Windows visibility beyond native logging.
- Built and administered an Active Directory domain environment.
- Simulated brute force attacks using Kali Linux and Hydra.
- Mapped Atomic Red Team activity to MITRE ATT&CK techniques.
- Investigated Windows security events using Splunk searches.
